# MedFlex — SaaS Dashboard Assignment

## Project Overview

A simple healthcare staffing dashboard layout built using **HTML** and **Tailwind CSS**. This project focuses on clean structure and responsive design for a frontend assignment, completely free of JavaScript.

## Features

* **Responsive Layout:** Automatically scales for mobile and desktop screens.
* **Hero Metrics:** Displays key data cards (Verified Locums, Partner Hospitals, etc.).
* **Placements Queue:** A clean list tracking active and pending hospital shifts.
* **Activity Canvas:** A placeholder section ready for future data metrics.

## Tech Used

* **HTML5** for page structure.
* **Tailwind CSS (via CDN)** for styling.
* **Font Awesome (via CDN)** for dashboard icons.

## How to Run

1. Download the `index.html` file.
2. Double-click the file to open it instantly in any web browser.
